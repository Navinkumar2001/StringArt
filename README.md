# String Art Visualizer

An interactive web application for visualizing and creating string art patterns.

## Features

- **Pin Sequence Visualization**: Load and navigate through pin sequences step by step
- **Image to String Art Conversion**: Upload any image and convert it to string art automatically
- **Interactive Controls**: Navigate through sequences, simulate steps, and generate full artwork
- **Customizable Parameters**: Adjust number of pins, lines, and line thickness

## Live Demo

Visit the live application: [String Art Visualizer](https://navinkumar2001.github.io/StringArt/)

## How to Use

1. **Manual Sequence**: Enter a comma-separated pin sequence in the textarea
2. **Image Upload**: Upload an image and let the algorithm generate string art automatically
3. **Navigation**: Use Previous/Next buttons or "Go to Line" to navigate through the sequence
4. **Visualization**: Watch the string art being created step by step or generate the complete artwork

## Technology

- Pure HTML5, CSS3, and JavaScript
- Canvas API for rendering
- Image processing algorithms for automatic string art generation

## Local Development

Simply open `index.html` in your web browser - no build process required!
